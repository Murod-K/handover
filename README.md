# 🏗 Handover Bot — Бот передачи смены

Telegram-бот для стандартизированной передачи смены на строительной площадке.

## Архитектура

```
handover_bot/
├── main.py                  # Точка входа, регистрация роутеров
├── config.py                # Конфигурация (токены)
├── enums.py                 # Все коды данных (ENUM)
├── states.py                # FSM состояния
├── data/
│   └── translations.py      # Словари 4 языков (RU/UZ/EN/TR)
├── handlers/
│   ├── start.py             # /start
│   ├── language.py          # Выбор языка
│   ├── project.py           # Выбор объекта / подобъекта
│   ├── shift.py             # Открытие смены
│   ├── structure.py         # Тип + название + натура
│   ├── rebar.py             # Арматура + дефекты
│   ├── concrete.py          # Бетон + готовность
│   ├── pump.py              # Насос + логистика
│   └── report.py            # Завершение + генерация отчёта
├── keyboards/
│   └── builders.py          # Все клавиатуры (из словарей)
├── repositories/
│   ├── user_repo.py         # Пользователи, язык
│   ├── project_repo.py      # Проекты
│   └── shift_repo.py        # Смены + записи
├── services/
│   ├── database.py          # Инициализация SQLite
│   ├── entry_service.py     # Сохранение записи
│   ├── report_service.py    # Генерация отчёта
│   └── translation_service.py # OpenAI перевод комментариев
├── requirements.txt
├── Dockerfile
└── docker-compose.yml
```

## Ключевые принципы

- **Все данные хранятся в кодах** (ENUM): `NATURA_GIVEN`, `REBAR_PARTIAL`, `POURING_MOBILE_PUMP`
- **Нет свободного ввода** кроме названия конструкции и комментариев
- **Перевод комментариев** — только через OpenAI, temperature=0, дословно
- **Отчёт из кодов** — никакого перевода текста, только словари
- **Блокировка заливки** если натура не дана или арматура не сдана

## Запуск

### Локально
```bash
pip install -r requirements.txt
python main.py
```

### Docker
```bash
docker-compose up -d
```

## Добавление проектов

Отредактируйте `services/database.py` — секцию `-- Seed projects`.

## Поток ввода (FSM)

```
/start
  └─> Выбор языка
    └─> Выбор объекта
      └─> Выбор подобъекта / блока
        └─> Тип смены (День / Ночь)
          └─> [Для каждой конструкции:]
                Тип конструкции (кнопка)
                └─> Название (текст)
                  └─> НАТУРА (кнопка)
                    └─> АРМАТУРА (кнопка)
                      └─> Дефекты (чек-лист, если ≠ сдана)
                        └─> БЕТОН (кнопка)
                          └─> [если заливка:]
                                Наличие бетона
                                Готовность (опалубка/гидроизол/арматура)
                                Объём м³
                                Метод подачи
                                └─> [если насос:]
                                      Тип насоса
                                Логистика насоса
          └─> Завершить смену → Отчёт
```

## Интеллектуальные блокировки

| Условие | Действие |
|---------|----------|
| Натура = НЕ ДАНА | ⚠️ Предупреждение + пометка в отчёте |
| Натура = БУДЕТ НОЧЬЮ | ⚠️ Предупреждение |
| Арматура = НЕ СДАНА + заливка | 🚫 Предупреждение о блокировке |
| Бетон отсутствует | 🔴 Критическое предупреждение |
